## Typography

| Token | Value |
|---|---|
| `--serif` | `'Playfair Display', Georgia, serif` |
| `--sans` | `'DM Sans', 'Helvetica Neue', sans-serif` |
| `--mono` | `'DM Mono', 'Courier New', monospace` |

**Google Fonts import:**
```
Playfair Display: ital,wght@0,400;0,600;1,400
DM Sans: wght@300;400;500
DM Mono: wght@300;400
```

### Usage Convention
| Font | Used for |
|---|---|
| Playfair Display | Section headings, card titles, editorial labels |
| DM Sans | Body text, nav, buttons, descriptions |
| DM Mono | Hex codes, data labels, timestamps, badges, eyebrows |

## Voice & Tone

- **Authoritative but not academic** — the site speaks like a senior practitioner
- **Systems-first** — everything connects to something else
- **No hype** — no "revolutionary" or "cutting-edge"; let the specificity do the work
- **Metric precision** — "12–18 day delay" not "significant delays"
