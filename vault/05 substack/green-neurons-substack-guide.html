01
Topbar
Auto-generated
Substack Path
Dashboard › Settings › Website → Header
01
Substack auto-generates the topbar. You cannot fully control it — it shows your publication name and subscribe/login links by default.
02
What you can do: Add social links in Settings → Social Links. They'll appear in the header area on some layout modes.
03
The issue date / volume number shown in the mockup is not native — approximate it by adding it to your publication tagline.
↳The topbar is intentionally minimal in Substack. Let it be — the masthead below is where your brand lives.

02
Masthead
Full Control
Substack Path
Dashboard › Website Editor › Home → Header
01
Go to Website Editor → Home → Header. Set Appearance to "Wordmark" (not Publication Name text).
02
Upload your horizontal Wordmark PNG (see asset specs →). This replaces the text wordmark entirely with your branded lockup.
03
Upload a square Logo PNG for the favicon / app icon. Keep it simple — the neural node mark works well at small sizes.
04
Set Header layout to "Stacked" — wordmark above nav. This gives the clean masthead → nav separation from the mockup.
↳Use a transparent PNG so the wordmark floats on the cream background cleanly. No white box behind it.

04
Hero — Featured Post
Full Control
Substack Path
Website Editor › Home › Hero → Magazine Mode
01
In Website Editor → Home → Hero, select layout mode: "Magazine". This gives you the large left-title / right-image split from the mockup.
02
Pin a post to the hero using the orange pin icon — click the pencil in the hero section while in the editor. The most recent post shows by default otherwise.
03
The hero pulls the post's title, subtitle, and cover image automatically. Write your post titles in the editorial style — the title IS the design here.
04
For the oversized "GAP"-style word: use a short, punchy subtitle. Substack renders the title large in Magazine mode — lean into it.
↳Title writing tip: lead with the tension, trail with the italicized concept. "The fabric intelligence GAP" — three words max in the kicker, then let the noun hit hard.

Index Row — 3 Pinned Posts
Partial Control
Substack Path
Website Editor › Home › Posts & Modules → Content Block
01
Enable Custom Body Layout in Website Editor → Home. The "Body" tab becomes "Posts & Modules".
02
Add a Content Block. Set display to "Grid", source to "Recent Posts", rows to 1. This gives you the 3-column index row.
03
To pin specific posts here (not just recent), set source to a specific Tag and tag those 3 posts with a "Featured" tag.
04
The numbered index (01, 02, 03) and the pure-type no-thumbnail look cannot be fully replicated in Substack — it will render with thumbnails by default.
⚠Workaround for pure-type look: Use a consistent dark navy square thumbnail (your brand color) for all posts in this row. Gives the impression of intentional blocks, not random photos.

06
Type Break — Dark Brand Block
Workaround Required
The Challenge
✕Substack has no native dark full-bleed content block. The type break cannot be replicated 1:1 with native tools.
Best Workaround — Subscribe Banner
Website Editor › Posts & Modules › Add → Subscribe Module
01
Add a Subscribe Module between your content blocks. It renders as a mid-page subscribe prompt — closer to the dark callout than anything else available.
02
Customize the subscribe module text: headline = "Where systems thinking meets the supply chain." CTA = "Subscribe Free".
03
The subscribe module inherits your accent color — set accent to #080c08 (near-black) for the dark treatment. This is the closest you'll get.
↳The ghost-outline text effect and Bebas Neue display type are only achievable in the HTML mockup, not on Substack. Accept this gracefully — the type break concept survives in the subscribe module's dark treatment.
Subscribe Module Copy
FieldCopy
HeadlineWhere systems thinking meets the supply chain.
SubheadTechnical apparel, wearable technology, production systems intelligence. Twice monthly. No noise.

Alternative — Pinned About Page
A
Create a standalone Page with your manifesto content. Add it as a Homepage Link block — this renders a custom card mid-page with your chosen image and copy.
B
Use a dark navy image (1200×800, #1a2535) as the page's cover — approximates the dark zone from the mockup.
07
Grid + Sidebar
Full Control
Substack Path — Grid
Website Editor › Posts & Modules › Add Content Block → Grid
01
Add a second Content Block below the subscribe module. Set display: Grid, rows: 2, source: Recent Posts. This is your 2×2 card grid.
02
Add a title override: "Recent Issues" with a see-all link pointing to your archive.
Substack Path — Sidebar
Settings › Website → Sidebar
03
Enable the Sidebar in Settings → Website → Sidebar. Substack shows it on the homepage and post pages.
04
Add Sidebar Links for each topic/section. These render as a clean link list — matches the Topics block from the mockup.
05
The subscribe box in the sidebar is auto-generated by Substack — you get it for free.
↳The sidebar is one of Substack's strongest native features. It persists across homepage and post pages — use it for navigation, not decoration.

08
List View + Footer
Full Control
Substack Path — List Block
Website Editor › Posts & Modules › Add Content Block → List
01
Add a third Content Block. Set display: List, source: Recent Posts, rows: 4. This is the numbered archive list at the bottom.
02
Title override: "Archive" with see-all link pointing to /archive.
03
The numbered index (01, 02…) is not native — Substack list mode shows title + thumbnail + date. The editorial feel comes from your titles and consistent thumbnails.
Substack Path — Footer
Settings › Website → Footer
04
In Settings → Website → Footer, set Footer background color to #080c08. This gives the dark footer from the mockup.
05
Enable Publication Name and Social Links in footer. Add a copyright line if the field is available.
↳The footer is minimal in Substack — you can't do the full 4-column layout from the mockup. But dark bg + name + socials is enough. Less is more here.
